# Info Bot

A **Telegram Info Bot** built with **Node.js**.  
This bot provides the following commands:

- `/start` - Welcome message with list of commands
- `/time` - Shows current time
- `/date` - Shows today’s date
- `/weather <city>` - Shows current weather for a city

✅ All commands are **case-insensitive**, e.g., `/start`, `/Start`, `/START` work.

---

## Features

- Easy-to-use Telegram commands
- Case-insensitive command handling
- Simple, clean Node.js code
- Fetches weather using the [wttr.in](https://wttr.in) API
- Friendly error handling if command is used incorrectly
- Secure token management with `.env`
